Elevations#

Elevation - это относительное расстояние между двумя поверхностями по оси z. Тени дают подсказки пользователю и помогают сориентироваться в иерархии интерфейсов. Доступно 25 уровней высоты. Elevation(n) где n - от 0 до 24.
