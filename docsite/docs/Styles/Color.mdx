## Contrast

Функция Color() принимает на вход цвет и определяет 
контрастный белый или черный цвет для лучшей читаемости текста. 
Функция возвращает объект с полями Color и Contrast в hex формате.

```js
Color('blue700')
return {
    Color: '#1976D2',
    Contrast: '#FFFFFF'
}
```
Форматы цвета которые может принимать функция Color() [NamePalettes/hex/rba(a)/hsl]
