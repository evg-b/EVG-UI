Эти цветовые палитры, изначально созданные [Material Design](https://material.io/design/color/the-color-system.html#tools-for-picking-colors) в 2014 году, состоят из цветов, гармонично сочетающихся друг с другом.

```js
import { colors } from '@evg-b/evg-ui';
colors.red['700'] // #D32F2F

или

const { red } from '@evg-b/evg-ui'
red['700'] // #D32F2F
```
