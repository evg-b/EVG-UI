Это маленький значок обычно используется в паре с другими компонентами. Используется для привлечения внимание к этим компонентам.

## Color

Цвет компонента меняется через свойство `color`. Более подробно про цвет в этой [статье](/docs/styles/color#props-color).

<BadgeColorExample />

## Max

Свойство `max={number}` определяет максимальный предел точного отображения.

<BadgeMaxExample />

## Dot

Если вы хотите минималистичный режим отображения, то используйте свойство `dot={true}`.

<BadgeDotExample />

## Circle

Свойство `сircle={true}` помогает корректно позиционироваться относительно цели круглой формы.

<BadgeCircleExample />

## Ripe

Свойство `ripe={true}`, badge становиться самостоятельным элементом.
